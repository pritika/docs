## Replacing images

The Image Editor (üñºÔ∏è icon) provides three ways to source and replace media for your site‚Äîwhether it‚Äôs a standard `<img>` element or a background image.

### File upload

- **Formats**: JPG, PNG, SVG
- **Max size**: 10MB
- **Performance**: Files over **0.5MB** are automatically compressed to help maintain site speed

### Unsplash search

Search and select professional stock photography directly within the editor. Selecting an image automatically handles the required photographer attribution.

### AI generation

Provide a text description (for example, ‚ÄúA modern office with natural lighting‚Äù) and the AI generates a unique, professional image for your site.

## Adjusting image filters and fit

Once an image is selected, the **Adjust** tab lets you fine-tune its appearance and how it sits within its container.

### Image fit modes

- **Cover**: Fills the container (image may be cropped).
- **Contain**: Shows the full image within the box (may leave empty gaps).
- **Fill**: Stretches the image to match the container dimensions.

### Visual filters

Use sliders to adjust:

- Brightness
- Contrast
- Grayscale
- Sepia
- Blur

If needed, use **Reset All Filters** to return to the original image state instantly.
