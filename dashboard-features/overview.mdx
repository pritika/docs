Welcome to the Butternut Dashboard Features Guide. This section covers the advanced tools available in your website dashboard.

## What you can manage here

- **Domains**: Link a custom domain and generate SSL.
- **Subscribers**: View and export newsletter subscribers.
- **SEO**: Control robots.txt, sitemap.xml, Google verification, and per-page metadata.
- **Analytics**: Connect Google Analytics and Google Tag Manager.
- **Custom chatbot**: Enable and train an AI chatbot for visitors.
- **Accept payments**: Configure PayPal, Stripe, or Razorpay (one active at a time).
- **Form data**: View/export submissions and configure notifications.
- **Custom scripts**: Add third-party scripts to Head or Body.
- **Contact us settings**: Choose where contact form submissions are emailed.

## Prerequisites (most features)

- Many dashboard tools require a **paid subscription**.
- Some tools also require **admin access**.

## Recommended setup order

1. Set **Contact Us** email
2. Connect **Google Analytics**
3. Link a **custom domain** and generate **SSL**
4. Configure **SEO metadata**
5. Enable **Subscribers** collection (newsletter section)
6. Set up **payments** (if selling)
7. Add **custom scripts** only if needed
